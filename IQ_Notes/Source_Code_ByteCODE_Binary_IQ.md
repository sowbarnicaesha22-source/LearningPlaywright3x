## 🔍 Concept Breakdown Table

| Layer | What It Is | Who Reads It | Human Readable? | Example |
|-------|-----------|--------------|-----------------|---------|
| **Source Code** | Code written by a developer in a high-level language | Developer / IDE | ✅ Yes | `console.log("Hello Testing Academy");` |

## ⚡ TL;DR

| Question | Answer |
|----------|--------|
| What is **Source Code**? | The human-readable code you write (e.g., `.js`, `.py`, `.java`) |
| What is **Bytecode**? | A platform-neutral intermediate representation processed by a runtime engine |
| What is **Binary Code**? | The final 0s and 1s executed directly by the CPU |
| Does JavaScript produce bytecode? | Yes — V8's **Ignition** interpreter compiles JS to bytecode internally |
| Does JavaScript run as binary? | Yes — **TurboFan** JIT-compiles hot bytecode paths into native machine code |
| Can you see JS bytecode? | Yes — run `node --print-bytecode your_file.js` in terminal |
